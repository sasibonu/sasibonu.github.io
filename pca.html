Prinicipal Component Analysis

Analysis and modeling can be made more difficult by the redundant or unnecessary features that are frequently present in high-dimensional data. By identifying the key structures and patterns in the data, dimensionality reduction aids in helping the user concentrate on the most important information. The analysis that follows is easier to handle and understand as a result of this reduction. Algorithms' computational complexity drops as dimensionality is reduced, which results in quicker execution times and more effective resource usage. Moreover, because of our limited senses, it is difficult or impossible for humans to directly visualize high-dimensional data. Data can be transformed into lower-dimensional environments by dimensionality reduction techniques, making them more effectively viewed. Comprehending intricate linkages and patterns becomes easier when data is projected into two or three dimensions.

Principal Component Analysis (PCA) is a popular method for dimensionality reduction and data compression in machine learning and data analysis. It functions by converting the initial high-dimensional data into a lower-dimensional space while keeping the majority of the data's variability. It is a way of simplifying data without losing it.

To begin with, the covariance matrix is computed, which is a matrix that includes estimates of the relationships between each variable in Z and all other variables in Z. It is quite effective to know how one variable is related to another. Two variables can have positive, negative, or absolutely no covariance between them.

Eigenvectors and Eigenvalues are the fundamental concepts of principal component analysis. What are eigen vectors and eigen values?

Eigenvectors are unique vectors whose magnitude changes but they continue to point in the same direction following transformation. While eigenvalues are only the constants that, when changed linearly, raise or reduce the eigenvectors along their span. The eigenvalues are sorted in descending order by PCA. The primary components are the eigenvectors that match the biggest eigenvalues. The directions of the data's largest variance are specified by these eigenvectors. In the end, PCA converts the original data into a new, lower-dimensional space by projecting it onto the primary components that were chosen.

High dimensions bring with them a lot of complications. High-dimensional data raises the danger of overfitting in machine learning models, which occurs when algorithms capture noise or unimportant patterns in the data instead of the underlying structure, along with lengthening computation times and decreasing intuition in visualizations. By emphasizing the most useful features and lowering the possibility of modeling noise, dimensionality reduction helps to mitigate overfitting.

Models and outcomes that are easier to understand are frequently produced by dimensionality reduction, which simplifies the data. The links between variables become more evident when the number of features is decreased, making it easier to evaluate the results of analyses and make predictions about the model.

PCA usually only requires numeric data. Although here textual data has been used, the data was transformed into high-dimensional numerical format using word2vec. This data was applied PCA on, which reduced the dimensionality and produced 2 principal components which was used in clustering.
